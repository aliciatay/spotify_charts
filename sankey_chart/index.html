Cross-Platform Performance Analysis

Understanding Cross-Platform Success

This parallel coordinates plot shows how songs perform across different streaming platforms. Each line represents a song, flowing through different platforms and showing its performance on each.

How to Read the Plot

  • Vertical Axes: Each axis represents a different platform (Spotify, YouTube, TikTok, etc.)
  • Lines: Each line is a song, showing its performance across all platforms
  • Height: Higher position on each axis means better performance on that platform
  • Performance Threshold: Red dashed lines show the cutoff point for the selected performance level (e.g., Top 25%)

Performance Filtering

  • All Songs: Shows all songs in the dataset
  • Top 25%: Highlights songs that maintain top 25% performance across platforms
  • Top 50%: Shows songs in the upper half of performance
  • Top 75%: Displays songs above the 25th percentile

Interactive Features

  • Hover over lines to highlight specific songs and see details
  • Use the genre selector to focus on specific music types
  • Use the performance selector to focus on top-performing songs
  • Red dashed lines show the threshold for each platform
